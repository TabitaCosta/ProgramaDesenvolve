O que é HTML ?

HTML, ou Linguagem de Marcação de Hipertexto, é a linguagem padrão utilizada para criar e estruturar páginas web. Ela define a estrutura do conteúdo, como textos, imagens, links e outros elementos, organizando-os de forma lógica para que os navegadores possam interpretá-los e exibi-los corretamente.

O que é CSS ?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML. Em outras palavras, o CSS é responsável por definir como os elementos de uma página web devem ser exibidos, como cores, fontes, layouts e outros aspectos visuais.

O que fullsatck ?

Em desenvolvimento de software, "full stack" refere-se a um desenvolvedor que possui conhecimento e habilidades para trabalhar tanto no front-end quanto no back-end de uma aplicação. Isso significa que ele pode lidar com a interface do usuário (o que o usuário vê e interage), bem como com a lógica do servidor, banco de dados e outras funcionalidades que ocorrem "por trás das cenas
